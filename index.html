LuciPHOr

Luciphor is a program that performs phospho-site localization on MS/MS data processed by the Trans-Proteomic Pipeline (TPP). It is unique in that it is the first phospho-site prediction program to provide estimates for the false localization rate (FLR).

Luciphor takes in as input PeptideProphet XML files and spectral files (mzXML, mzML). From these it predicts the correct phosopho-peptide permutation for the peptide sequences reported in the XML files.

Luciphor is written in C++ and uses the Boost and ProteoWizard libraries for reading mz*ML files. Instructions for compiling Luciphor locally on your computer are provided in the building_luciphor.html file. This software is designed to be compiled and executed on Linux. All testing has been performed on RHEL 6.3 (64-bit), GCC compiler version 4.4.6 20120305.
