Compiling Luciphor with Proteowizard

This tutorial covers how to compile Luciphor on Linux. The instructions are written assuming you are using RHEL/CENTOS 6.3 or Fedora 17. I don't use Ubuntu or any of the other flavors of Linux but I imagine these steps are generally applicable to those distributions.

  1. Download the Luciphor source code from this site: http://dfermin.github.com/luciphor
    For this tutorial we'll assume you downloaded the tar.gz file.
  2. Unpack the tarball. This should produce a folder named something like dfermin-luciphor-<d0f3388>
    The part that reads something like d0f3388 changes with any updates to github so this part can be different in your download.
  3. Unpack and prep the Proteowizard library for compilation. The following commands should be executed in order from within the dfermin-luciphor-<d0f3388>
    mkdir pwiz_root
    mv pwiz-src-without*.bz2 pwiz_root/
    mv boost_1_43_0.tar.bz2 pwiz_root/
    cd pwiz_root
    tar xjf pwiz_src_without*.bz2
    tar xjf boost_1_43_0.tar.bz2
  4. Now we will compile Proteowizard. WARNING: This will take a long time so be patient!
    Within the pwiz_root folder, type:
    bash quickbuild.sh pwiz/data/msdata

    If you have a computer with multiple processors you can speed up the process by typing:
    bash quickbuild.sh pwiz/data/msdata -j<THREADS>
    Where <THREADS> is the number of processors you want to use for building Proteowizard.

    Now back out of the pwiz_root folder. You're done with the hardest part of this ordeal.
    cd ..
  5. Assuming you compiled Proteowizard as described above. You should be able to generate the luciphor executable by running:
    bash compile.sh

    Like Proteowizard you can specify the number of threads to use for compilation:
    bash compile.sh -j<THREADS>

    Compilation will take some time so be patient!
    If all went well, you should find an executable program called luciphor in the current directory.
    For general usage instructions, execute luciphor and it will print out some (hopefully) useful information.
